Project Description:
This project models a Subscription Box Service database system. 
The system includes customers, subscriptions, subscription plans, products, and payments. 
The database was designed using Chen and UML ER diagrams and implemented in MySQL Workbench.

Files in this Repository:
1. ER_Diagrams Folder:
   - Chen_ER_Diagram.jpg (Hand-drawn Chen ER Diagram)
   - UML_ER_Diagram.png (UML ER Diagram from MySQL Workbench)

2. SQL Folder:
   - SubscriptionBoxDB.sql (SQL script to create the database and tables)

3. MySQL_Workbench Folder:
   - SubscriptionBoxDB.mwb (MySQL Workbench model file)

4. Report Folder:
   - SubscriptionBox_Project_Report.pdf (Final project report)

This repository contains all required materials for the CIS 344 individual project submission.
